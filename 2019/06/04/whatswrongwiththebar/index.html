为什么会多出一个小长条？

2019/06/04 | 小提示

博客总访问量：

如果你发现上面这个长条，你会发现：为什么每条博客上会多出一个小长条？

其实，这个是总博客的访问量组件。这个小长条，统计我们社整个博客的访问量。

这个小长条，其实就是hit-counter。

当然，如果你想将这个小长条挂在你的项目主页或者你的博客里，其实很简单！

你只要点击该链接，进入该网站后，你就会见到以下界面：

在“Generate SVG Code For A Different Website”这段文字下面的输入框里：

输入你的项目主页的地址或者你的博客的地址。页面会自动生成SVG。

生成完毕后，会有两个按钮：“Copy Image Tag”和“Copy Markdown”

如果你的博客是HTML的，你就点击“Copy Image Tag”，黏贴到你的HTML博客里面去。

如果你的博客是HEXO，或者黏贴到项目GitHub主页里的“README.md”文件，你就选择“Copy Markdown”，黏贴到你的HEXO博客，
或者是你的项目主页里的“README.md”文件。

最后，你的博客或者项目说明里就有这个小长条啦！
